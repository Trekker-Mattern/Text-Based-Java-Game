# Trekkers Text Based Java Game

## Overview

Trekkers Text Based Java Game is a text-based RPG built in Java. The game is centered on story, exploration, and player choice, with the experience delivered entirely through written interaction.

## Basis of the Game

The foundation of the game is based on a few simple ideas:

- **Text-based gameplay**: The player reads descriptions and responds with choices.
- **Branching decisions**: Choices influence the path the player takes through the game.
- **RPG elements**: The game includes basic role-playing features such as progression and encounters.
- **Roguelike elements**: The game includes vast replayability with many ways to play.
- **Java implementation**: The project is written completely in Java.
- **Learning experience**: The game is being developed as a way to learn and improve programming skills.

## Project Goal

The goal of the project is to create a playable game from scratch while building experience with game design, logic, and Java development. It is a way to showcase things learned both in university as well as from experience in projects large and small.
